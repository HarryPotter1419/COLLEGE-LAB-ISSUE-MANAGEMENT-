Actor

Students can create an account and securely log in to the system using their credentials.

Report Lab Issue

Submit detailed issue reports with images, descriptions, lab location, and system number.

UC6

Assign Issues to Technicians

Delegate specific issues to registered technicians for faster and more efficient resolution.

Update Status & Add Remarks

Actor
UC9

Technicians log in to view their personalized dashboard showing only issues assigned to them.

UC10

View Assigned Issues

UC11
